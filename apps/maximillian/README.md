## Maximillian Creative – Digital Design Studio

Premium design and creative strategy website for maxjj.co.za.

### Key files
- `index.html` — main portfolio site
- `404.html` — custom not-found page

### Features
- Showcase of design portfolio and case studies
- Services overview
- Creative process documentation
- Client testimonials
- Contact section

### Run locally
Open `index.html` in a browser, or serve with a simple HTTP server:
```bash
python3 -m http.server 8000
```
Then visit http://localhost:8000

### Customizations from FlowRoll
- **Color Scheme**: Deep red accent (#c41e3a) instead of blue
- **Branding**: Maximillian Creative instead of FlowRoll Supply
- **Content**: Digital design services vs. jiu-jitsu products
- **Visual Style**: Warm, sophisticated aesthetic for creative brand
